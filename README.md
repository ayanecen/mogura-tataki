# おしゃれでかわいいモグラ叩き 🐹✨

パステルカラーで統一された、おしゃれでかわいいモグラ叩きゲームです。
GitHub Pagesでそのまま動作し、モバイルでも快適に遊べます。

## ✨ 特徴
- **デザイン**: ガラスモーフィズムを採用したモダンでかわいいUI
- **モグラ**: CSS/SVGのみで描画された愛らしいキャラクター
- **難易度**: かんたん・ふつう・むずかしいの3段階
- **サウンド**: Web Audio APIによるリアルタイム生成（音声ファイル不要）
- **レスポンシブ**: PCはもちろん、スマホでも片手で遊びやすい設計

## 🚀 遊び方
1. [公開URL] にアクセス、または `index.html` をブラウザで開きます。
2. 「ゲームスタート」ボタンを押すと開始します。
3. 出てくるモグラをタップ（またはクリック）して叩きましょう。
4. 60秒間でどれだけ高いスコアを出せるか挑戦！

### 操作方法
- **タップ/クリック**: モグラを叩く
- **Enterキー**: ゲームの開始・再開

## 🔧 カスタマイズ
`game.js` の冒頭にある `DIFFICULTY` 定数を変更することで、ゲームバランスを調整できます。

```javascript
const DIFFICULTY = {
    easy: {
        spawnInterval: 1500, // モグラが出る間隔 (ms)
        moleDuration: 1200,  // モグラが出ている時間 (ms)
        maxMoles: 1          // 同時に出る最大数
    },
    // ...
};
```

## 🔈 音声について
ブラウザのポリシーにより、音声はユーザーの最初の操作（スタートボタンのクリック等）の後に有効化されます。
サウンドがONになっていても、操作前は鳴らない仕様ですのでご了承ください。

## 💻 開発・実行
特別なビルドツールは不要です。
リポジトリをクローンして `index.html` を開くだけで実行できます。

```bash
git clone [repository-url]
cd mogura-tataki
# あとは index.html を開くだけ！
```
