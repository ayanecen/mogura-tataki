# モグラ叩きゲーム 🎮

シンプルで楽しいモグラ叩きゲームです！HTMLとJavaScriptだけで作られているので、ブラウザがあればどこでも遊べます！

## 🎯 ゲームの特徴

- 60秒のタイマー付き
- スコア表示機能
- リトライボタン
- シンプルで見やすいデザイン
- モバイル対応
- 外部ライブラリ不要

## 🚀 遊び方

1. 画面上部に表示されるモグラをクリックします
2. モグラを叩くと10点獲得！
3. 60秒の間にできるだけ多くのモグラを叩きましょう
4. 時間切れになるとスコアが表示されます
5. 「もう一度プレイ」ボタンでリトライできます

## 🔧 開発環境のセットアップ

1. リポジトリをクローン
```bash
git clone [リポジトリのURL]
```

2. `index.html` をブラウザで開く
```bash
# ローカルで実行する場合
open index.html
```

## 🌐 GitHub Pagesでの公開手順

1. リポジトリの作成
   - GitHubにログイン
   - 新しいリポジトリを作成

2. コードのアップロード
   - `index.html` をアップロード
   - コミットを作成

3. GitHub Pagesの設定
   - リポジトリの「Settings」→「Pages」
   - Sourceを「main」ブランチに設定
   - 保存

4. 公開URLの確認
   - 数分待つとURLが表示されます

## 🎨 カスタマイズのヒント

- モグラの画像を変更する場合は、CSSの `.mole` クラスを編集
- ゲーム時間は `timeLeft` 変数で調整可能
- スコアの加算値は `score += 10` の部分で変更可能
- モグラの表示時間は `setTimeout` の値で調整可能
